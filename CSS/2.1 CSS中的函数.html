attr 函数用于获取被选元素的属性值。 它接受三个参数，属性名称、类型和默认值。 一个示例用法是用纯CSS实现tooltip功能。

counter()很早之前，计数器仅存在于ul，ol等元素中，如何想给其他元素增加计数，就只能通过list-style-image，或者background-image来实现。不过现在css3增加了counter函数，可以实现任何元素的计数作用。该函数与counter-reset以及counter-increment一起完成功能。
