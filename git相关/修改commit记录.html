问题

比如我们在提交代码的时候执行了git commit -m '提交代码'，但是我们又想修改提交代码为bug修复，这个时候我们应该怎么操作？

解决：

1、git log 查看提交记录 然后 q退出 ，执行git commit --amend

2、然后按一下键盘的 i 键，进入插入模式

3、修改描述为bug修复，然后按 esc 退出，在英文模式输入法下再按 shift + :，输入 wq

撤销本地commit git log 查看hash git reset [hash] 撤销本地commit到指定版本 切换分支 在新的分支上commit 提交 即可 取消变基 git rebase --abort
