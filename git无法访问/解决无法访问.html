首先去站长之家寻找一个代理
        http://tool.chinaz.com/dns?type=1&host=youzan.github.io&ip=

        进入网站后 在DNS查询tab输入你要查询的代理（比如youzan.github.io无法访问）：youzan.github.io
        复制一个代理ip

         第二步:找到hosts文件  必须以管理员的身份打开

        win10 hosts文件默认路径：C:\Windows\System32\drivers\etc

        这里教你如何以管理员身份打开hosts文件 shell -> notepad hosts

        然后 在文件后面
        添加：
            185.199.109.153    youzan.github.io
            185.199.109.153   elemefe.github.io
